# 🌤️ Responsive Weather App

A modern, responsive weather application built with Vanilla HTML, CSS (Glassmorphism design), and JavaScript. The app fetches real-time meteorological data from the **OpenWeatherMap API** to display weather updates globally.

## ✨ Features

- **Real-time Data**: Displays live temperature, "Feels Like" comfort level, and humidity.
- **16-Point Wind Trajectory**: Tracks wind speed and converts numerical degrees into a clear destination target (e.g., _"Towards North-West"_).
- **Dynamic Icon Rotation**: The wind arrow icon (`air.svg`) automatically rotates using CSS transforms to visually point in the direction the wind is traveling.
- **Glassmorphic UI**: Uses clean, frosted-glass background boxes that look stunning on any background.
- **100% Responsive**: Built using modern CSS Grid, Flexbox, and fluid typography (`clamp()`) to adapt flawlessly across mobile, tablet, and desktop viewports.
- **Sun Cycle Tracking**: Displays local sunrise and sunset operational times.

## 🔧 Installation & Local Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   ```
2. **Configure your API Key:**
   - Open `script.js`.
   - Replace the placeholder string in `apiKey` with your own active key from OpenWeatherMap.
3. **Run the app:**
   - Simply open the `index.html` file directly in any modern web browser.
